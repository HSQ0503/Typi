# Typi Privacy Policy

_Last updated: May 2, 2026_

Typi is a Chrome extension that helps users play back their own drafts into Google Docs with human-paced typing, pauses, corrections, and revisions.

## Data Typi stores locally

Typi stores the following data in Chrome's local extension storage on your device/browser profile:

- Your OpenAI API key, if you choose to save one
- Your selected typing speed
- The most recent typing plan and job status
- Recent error messages used for troubleshooting

This data is stored locally in your Chrome profile. Typi does not operate its own backend server.

## Data sent to OpenAI

When you start a typing plan, Typi sends the draft text you provide to OpenAI's API to generate a typing plan. The plan may include write actions, pauses, typo corrections, and revisions. Typi uses this response to type into the Google Doc you selected.

Typi does not send your draft text to any server controlled by Typi. OpenAI's handling of API data is governed by OpenAI's applicable policies and terms.

## Google Docs access

Typi runs on Google Docs pages so it can type into the document selected by the user. Typi does not read all of your Google Drive files. It interacts with the active Google Docs page for the purpose of writing the text you requested.

## Data sharing and selling

Typi does not sell user data. Typi does not share user data with advertisers or data brokers.

## Permissions

Typi requests the minimum permissions needed for its functionality:

- `storage`: saves local settings, API key, recent typing plan, job status, and errors.
- `activeTab`: lets Typi work with the Google Doc tab selected by the user.
- `https://docs.google.com/*`: allows Typi to type into Google Docs.
- `https://api.openai.com/*`: allows Typi to request typing-plan generation from OpenAI.

## Responsible use

Use Typi only with text you own or have permission to submit, publish, or enter into a document. Users are responsible for complying with applicable academic, workplace, and platform policies.
