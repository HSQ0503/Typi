How to get an OpenAI API key
  1. Open platform.openai.com/signup and create or sign in to your OpenAI account.
  2. Open billing at platform.openai.com/settings/organization/billing/overview and add a payment method if needed.
  3. Go to platform.openai.com/api-keys.
  4. Click Create new secret key, name it Typi, then copy the key.
  5. Paste it here and click Set API key for this user.

Typi stores the key locally in this Chrome profile and uses it only to request typing plans.
